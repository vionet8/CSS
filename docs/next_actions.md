## 帰宅後すぐにやること（30分）

### ① Node.js をインストールする

外出中にバックグラウンドでインストール試みたが、環境によっては手動が必要。

```powershell
# winget でインストール（管理者不要）
winget install OpenJS.NodeJS.LTS

# インストール確認
node --version   # → v22.x.x
npm --version    # → 10.x.x
```

公式サイト: https://nodejs.org/en （LTS 版を選択）

---

### ② フロントエンドの依存関係インストール

```powershell
cd C:\Apps\CSS\frontend
npm install
```

---

### ③ Anthropic API キーを設定

```powershell
notepad C:\Apps\CSS\backend\.env
```

`your_anthropic_api_key_here` を実際のキーに書き換える。  
キーの取得: https://console.anthropic.com/

---

### ④ 起動して動作確認

**ターミナル 1（バックエンド）**:
```powershell
cd C:\Apps\CSS\backend
.\.venv\Scripts\python.exe -m uvicorn app.main:app --reload --port 8000
```

**ターミナル 2（フロントエンド）**:
```powershell
cd C:\Apps\CSS\frontend
npm run dev
```

ブラウザで `http://localhost:5173` を開く。

## 判断が必要な設計ポイント

### エクスポート形式

スライドをどのフォーマットで出力するか:
- **PDF**: シンプル・印刷対応（html2canvas + jspdf）
- **PPTX**: 本格的だが実装複雑（python-pptx）
- **HTML**: Revealjs 形式でプレゼン可能

→ **推奨**: まず PDF を実装してから PPTX を検討

### Whisper のモデルサイズ

| モデル | 精度 | 速度 | RAM |
|--------|------|------|-----|
| tiny | 低 | 超速 | 1GB |
| base | 中 | 速い | 1GB |
| small | 高 | 普通 | 2GB |
| medium | 非常に高 | 遅い | 5GB |

→ **推奨**: `small`（日本語精度と速度のバランスが良い）
